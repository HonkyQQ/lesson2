# Подсказки по командной строке

Команда смены дирректории 
```sh
cd c:\folder_name
```

Команда отображения текущей дирректории
```sh
pwd
```

Листинг текущей дирректории 
Windows: 
```sh
dir
```
Linux, MacOS:
```sh
ls
```

Удаление файла в Windows:
```sh
del <filename>
```
Linux, MacOS:
```sh
rm <filename>
```
